# Introduction

Bienvenue sur la documentation de BetterEcoleDirecte ! Ce site sert de base de connaissance pour rassembler toutes les informations et ressources nécessaire
au développement du projet.

## Qu'est ce que BetterEcoleDirecte ?

BetterEcoleDirecte est un projet visant à reproduire EcoleDirecte avec pleins de fonctionnalités en plus, telles qu'une meilleure interface plus simple d'utilisation, ...

[En savoir plus](https://better-ecoledirecte.github.io/dev-blog) sur le projet BetterEcoleDirecte.

## Où se trouve le projet ?

Le projet est hébergé sur [Github](https://github.com/better-ecoledirecte). Il est composé de plusieurs dépôts :

* [Client](https://github.com/better-ecoledirecte/client)
* [API](https://github.com/better-ecoledirecte/api)
* [Documentation](https://github.com/better-ecoledirecte/docs)
* [Dev blog](https://github.com/better-ecoledirecte/dev-blog)

## Comment contribuer ?

Pour contribuer au projet [BetterEcoleDirecte](https://github.com/better-ecoledirecte), vous devez faire un [fork](https://docs.github.com/en/get-started/quickstart/fork-a-repo) 
sur votre compte GitHub du dépôt qui vous intéresse. Une fois les modifications faites, vous devrez faire une Pull Request qui sera analysée.
