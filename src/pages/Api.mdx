# Api

Il a été décidé qu'il sera fait une librairie complète afin de récupérer les informations nécessaires au bon fonctionnement de BetterEcoleDirecte sur l'api officielle d'EcoleDirecte.

Librairie qui sera réalisée en Dart afin de faciliter son utilisation depuis le Client de BetterEcoleDirecte, réalisé en Dart avec Flutter.

## Où se trouve la librairie ?

Elle est hébergée sur [Github](https://github.com/better-ecoledirecte/api).

## Comment contribuer ?

Pour contribuer à la librairie [api](https://github.com/better-ecoledirecte/api) de BetterEcoleDirecte, vous devez faire un [fork](https://docs.github.com/en/get-started/quickstart/fork-a-repo) 
sur votre compte GitHub du dépôt. Une fois les modifications faites, vous devrez faire une Pull Request qui sera analysée.
